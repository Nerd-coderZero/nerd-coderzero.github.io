Breast Cancer Detection via Hybrid CNN-Transformer

Hybrid architecture combining EfficientNet-B3 convolutional feature extraction with Transformer cross-attention for histopathology image classification. Architecture EfficientNet-B3 → patch tokens → cross-attention → classifier
Dataset BreakHis — 7,909 images · 4 magnifications · benign/malignant
